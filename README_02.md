# 02 截图接单优化独立版

这个子项目从 `C:\Users\Owner\Desktop\1` 复制出来，但后续修改只在 `02` 内进行，不影响主项目。

## 本版规则

- `选择列表截图` 只负责选择文件、显示文件名和预览图。
- `第一步识别` 才开始切订单卡、OCR、提取订单资料。
- 列表截图识别完成后只显示订单表格，不自动做地图定位、粗推荐或接单分析。
- 确认资料无误后，再按 `✓` 才进入接单推荐分析。

## 运行方式

如果直接用浏览器打开 `index.html`，OCR 和地图代理需要本地服务。

运行：

```powershell
C:\Users\Owner\Desktop\1\02\run-server.ps1
```

然后打开：

```text
http://127.0.0.1:8788/
```

主项目仍然可以继续使用 `http://127.0.0.1:8787/`。
